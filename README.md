# Hazeflow_Synthetic_Haze_Generator
It is a synthetic haze generation framework built on HazeFlow, integrating Depth Anything V2, MCBM beta maps, and the Atmospheric Scattering Model. It supports multi-GPU inference, configurable haze synthesis, automatic evaluation (PSNR, SSIM, MSE, MAE, FPS), CSV reporting, and large-scale dataset generation.
